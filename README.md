Para Meu Amor

Um projeto simples e especial feito com **HTML, CSS e JavaScript** para demonstrar carinho e amor através de uma página personalizada.

Funcionalidades

* Mensagem romântica personalizada
* Foto especial do casal
* Botão com declaração de amor
* Animação de corações caindo pela tela
* Link para a música "Until I Found You"
* Design responsivo para celular e computador

Prévia

A página exibe:

* Título romântico
* Mensagem especial
* Foto do casal
* Botão interativo
* Música do casal
* Efeito visual com corações animados

Tecnologias Utilizadas

* HTML5
* CSS3
* JavaScript

Estrutura do Projeto

```bash
projeto/
│
├── index.html
├── nos.jpeg
└── README.md
```

Como Executar

1. Baixe ou clone este repositório:

```bash
git clone https://github.com/SEU-USUARIO/SEU-REPOSITORIO.git
```

2. Abra a pasta do projeto.

3. Clique duas vezes no arquivo:

```bash
index.html
```

ou abra utilizando o navegador.

Objetivo

Este projeto foi criado como uma homenagem para uma pessoa muito especial, utilizando programação para transformar sentimentos em uma experiência visual simples e significativa.
